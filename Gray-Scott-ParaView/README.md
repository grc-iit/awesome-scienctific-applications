# Gray-Scott ParaView Visualization

## Overview

This application provides ParaView-based visualization and in-situ analysis for the Gray-Scott reaction-diffusion simulation. Using ADIOS2 as the data transport layer, it enables real-time visualization of the evolving concentration fields during simulation, as well as post-hoc analysis of saved output data. ParaView's VTK pipeline is used to render 2D and 3D visualizations of the reaction-diffusion patterns.

## Key Features

- **In-situ visualization**: Real-time rendering of Gray-Scott simulation data via ADIOS2 SST (Sustainable Staging Transport) engine
- **Post-hoc analysis**: Visualization of ADIOS2 BP files generated by the Gray-Scott simulation
- **Volume rendering**: 3D visualization of concentration fields using ParaView's volume rendering capabilities
- **Contour extraction**: Isosurface extraction for identifying pattern boundaries
- **Time series analysis**: Animated visualization of pattern evolution over simulation time steps

## Architecture

```
Gray-Scott Simulation  --(ADIOS2 SST/BP)-->  ParaView VTK Pipeline  -->  Rendered Output
```

The visualization pipeline connects to the Gray-Scott simulation either:
1. **In-situ (SST engine)**: Directly streams data from the running simulation
2. **Post-hoc (BP engine)**: Reads data files produced by a completed simulation

## Dependencies

- ParaView (5.10+)
- VTK
- ADIOS2 (with SST engine support)
- Python 3 (for ParaView Python scripting)
- MPI library
- CMake (build system)

## Build Instructions

```bash
mkdir build && cd build
cmake .. -DParaView_DIR=/path/to/paraview -DADIOS2_DIR=/path/to/adios2
make -j
```

## Usage

### In-situ Visualization

Run the Gray-Scott simulation and the ParaView visualization simultaneously:

```bash
# Terminal 1: Start Gray-Scott simulation with SST output
mpirun -n 4 ./gray-scott settings.json

# Terminal 2: Start ParaView visualization reader
mpirun -n 1 ./pdf_calc gs.bp plot.bp
```

### Post-hoc Visualization

Open the ADIOS2 BP output files in ParaView:

1. Launch ParaView
2. Open the `.bp` file directory
3. Apply volume rendering or contour filters to the U or V concentration fields

### Python Scripting

```python
from paraview.simple import *

reader = ADIOSReader(FileName='gs.bp')
display = Show(reader)
display.SetRepresentationType('Volume')
ColorBy(display, ('POINTS', 'U'))
Render()
```

## Configuration

The ADIOS2 XML configuration file controls data transport:

```xml
<adios-config>
  <io name="SimulationOutput">
    <engine type="SST"/>
  </io>
  <io name="VisualizationInput">
    <engine type="SST"/>
  </io>
</adios-config>
```
